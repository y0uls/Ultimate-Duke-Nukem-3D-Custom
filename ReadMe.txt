A huge thank you to all the contributors, map creator, community, mod developer and so on, for allowing me the creation of this launcher, bringing together all the best custom maps from this iconic game, Duke Nukem 3D!

To use it, simply run the "Ultimate Duke Nukem 3D Custom.bat" batch and choose the launch options!
Have fun for many hours!

PS : If you want to start the game in fullscreen you will have to go through a CMD window and execute the script : "Ultimate Duke Nukem 3D Custom.bat" --fullscreen OR Or change "wt --fullscreen -d . cmd /k "CustomDuke\udnc.bat" %1" in "Ultimate Duke Nukem 3D Custom.bat" to "wt --fullscreen -d . cmd /k "CustomDuke\udnc.bat" --fullscreen"
